## REGRAS DA API

## Usuários - Regras

* Email único;
* Senha com mais de 6 caracteres;
* Grupo padrão para todos os usuários;
* Cada usuário pode ter seu próprio grupo de contatos.

## Usuários - Validações

* Validar campo email;
* Validar campo senha;
* Validar campo nome;
* Validar se email já está cadastrado;
* Validar senha de usuário;
